# Codveda Data Science
This repository contains the work completed during my Data Science Internship at Codveda Technology, organized across six structured tasks spanning three difficulty levels. The project covers core data science disciplines including data cleaning and preprocessing, exploratory data analysis, linear regression, classification, time series analysis, and NLP-based text classification. Datasets used include real-world Telecom Churn data, the classic Iris dataset, Boston Housing prices, and Social Media Sentiment data. Each task is implemented as a standalone Python script with strict library constraints, paired with visualizations and compiled into a final internship report.
